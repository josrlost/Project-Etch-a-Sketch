# Project-Etch-a-Sketch
This is my attempt at the TOP Project: Etch-a-Sketch in the Foundation module. The objective of the assignment was to create a sizable grid, in which each cell reacts to the user's mouse potion and changes its color randomly, creating the illusion of "coloring" a white canvas. For this exercise I utilized multiple loops: a main loop to generate the div containers, and add them into an array; each array element would then be appended into a parent div container; and then get its respective css properties through JavaScript on said loop. 
I created an event listener on a button that on the click event prompts the user to resize the grid, by executing a function that validates the user's input, and triggers an array based on the input validation, which then generates a array of div nodes. For the Mouseover and Mouseout events I utilized the forEach method on the node arrays, to pass down the event to each array's element.
